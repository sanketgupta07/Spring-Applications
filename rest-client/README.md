# REST Client

### Service Description

URI: http://gturnquist-quoters.cfapps.io/api/random

It randomly fetches quotes about Spring Boot and returns them as a JSON document.

Sample Output:

	{
	   type: "success",
	   value: {
	      id: 10,
	      quote: "Really loving Spring Boot, makes stand alone Spring apps easy."
	   }
	}
